> **Professional Python trading bot for Polymarket prediction markets with gasless transactions, real-time WebSocket orderbook streaming, and automated trading strategies.**

🚀 Overview
----------

The **Polymarket Trading Bot** is a production-ready Python library and CLI tool for automated trading on Polymarket, the leading decentralized prediction market platform. Built for traders and developers who need reliable, low-latency access to Polymarket's orderbook and trading infrastructure.

### Key Capabilities

* ⚡ **Zero Gas Fees** - Builder Program integration for gasless transactions
* 📊 **Real-time Data** - WebSocket orderbook streaming with sub-second latency
* 🤖 **Automated Strategies** - Pre-built flash crash detection and custom strategy framework
* 🔒 **Enterprise Security** - PBKDF2 + Fernet encrypted key storage
* 🎯 **15-Minute Markets** - Native support for BTC, ETH, SOL, XRP prediction markets
* 🛠️ **Developer Friendly** - Clean Python API with comprehensive documentation

💻 Usage Examples
----------------


### Basic Bot Initialization


```
from polymarket_arbitrage import create_bot_from_env
import asyncio

async def main():
    # Initialize bot from environment variables
    bot = create_bot_from_env()
    
    # Get open orders
    orders = await bot.get_open_orders()
    print(f"Open orders: {len(orders)}")
    
    # Get trade history
    trades = await bot.get_trades(limit=10)
    for trade in trades:
        print(f"Trade: {trade['price']} @ {trade['size']}")

asyncio.run(main())
```

### Place a Limit Order


```
from polymarket_arbitrage import TradingBot, Config

# Initialize with configuration
config = Config(safe_address="0xYourProxyWallet")
bot = TradingBot(config=config, private_key="0xYourPrivateKey")

# Place a buy order
result = await bot.place_order(
    token_id="123456789012345678901234567890",
    price=0.65,  # 65% probability
    size=10.0,   # $10 USDC
    side="BUY"
)

if result.success:
    print(f"Order placed: {result.order_id}")
else:
    print(f"Order failed: {result.message}")
```

### Real-time WebSocket Streaming


```
from polymarket_arbitrage import MarketWebSocket

async def main():
    ws = MarketWebSocket()
    
    # Set up orderbook update handler
    @ws.on_book
    async def handle_book(snapshot):
        print(f"Asset: {snapshot.asset_id}")
        print(f"Mid Price: {snapshot.mid_price:.4f}")
        print(f"Best Bid: {snapshot.best_bid:.4f}")
        print(f"Best Ask: {snapshot.best_ask:.4f}")
    
    # Subscribe to token IDs
    await ws.subscribe(["token_id_1", "token_id_2"])
    
    # Run WebSocket connection
    await ws.run(auto_reconnect=True)

asyncio.run(main())
```

### Custom Trading Strategy


```
from polymarket_arbitrage import TradingBot
from polymarket_arbitrage.strategies import BaseStrategy, StrategyConfig
from polymarket_arbitrage.core.websocket_client import OrderbookSnapshot

class MyStrategy(BaseStrategy):
    async def on_book_update(self, snapshot: OrderbookSnapshot) -> None:
        """Handle orderbook updates."""
        if snapshot.mid_price < 0.30:
            await self.execute_buy("up", snapshot.mid_price)
    
    async def on_tick(self, prices: dict) -> None:
        """Called on each strategy tick."""
        # Your trading logic here
        pass
    
    def render_status(self, prices: dict) -> None:
        """Render status display."""
        print(f"UP: {prices.get('up', 0):.4f} | DOWN: {prices.get('down', 0):.4f}")

# Initialize and run
bot = TradingBot(...)
config = StrategyConfig(coin="ETH", size=5.0)
strategy = MyStrategy(bot, config)
await strategy.run()
```

📈 Trading Strategies
--------------------


### Flash Crash Strategy

The Flash Crash strategy monitors 15-minute prediction markets for sudden probability drops and executes trades automatically.

**How It Works:**

1. Monitors real-time orderbook prices via WebSocket
2. Tracks price history over configurable lookback window
3. Detects when probability drops exceed threshold
4. Executes buy orders on crashed side
5. Manages positions with take-profit and stop-loss

**Usage:**

```
# Default settings (ETH, $5 size, 30% drop threshold)
polymarket-flash-crash --coin ETH

# Custom configuration
polymarket-flash-crash \
    --coin BTC \
    --size 10.0 \
    --drop 0.30 \
    --lookback 10 \
    --take-profit 0.10 \
    --stop-loss 0.05
```

**Parameters:**

| Parameter | Description | Default |
| --- | --- | --- |
| `--coin` | Coin symbol (BTC, ETH, SOL, XRP) | ETH |
| `--size` | Trade size in USDC | 5.0 |
| `--drop` | Drop threshold (absolute change) | 0.30 |
| `--lookback` | Detection window in seconds | 10 |
| `--take-profit` | Take profit in dollars | 0.10 |
| `--stop-loss` | Stop loss in dollars | 0.05 |
| `--debug` | Enable debug logging | False |

🔐 Gasless Trading Setup

Enable zero-fee transactions via Polymarket Builder Program:

🔒 Security
----------


### Private Key Encryption

Private keys are encrypted using industry-standard cryptography:

* **Key Derivation**: PBKDF2-HMAC-SHA256 with 480,000 iterations
* **Encryption**: Fernet (AES-128-CBC + HMAC-SHA256)
* **Salt**: Unique 32-byte salt per encryption
* **File Permissions**: Automatically set to 0600 (owner-only)

### Best Practices

* ✅ Never commit `.env` files or private keys to version control
* ✅ Use a dedicated trading wallet separate from main funds
* ✅ Store encrypted key files in secure locations
* ✅ Use strong passwords (12+ characters, mixed case, numbers, symbols)
* ✅ Rotate keys periodically
* ✅ Review and audit code before using with real funds

### Encrypting Your Private Key


```
from polymarket_arbitrage.core import KeyManager

manager = KeyManager()
manager.encrypt_and_save(
    private_key="0xYourPrivateKey",
    password="your_secure_password",
    filepath="credentials/key.enc"
)
```

🐛 Troubleshooting
-----------------


### Common Issues


| Issue | Solution |
| --- | --- |
| **Missing credentials** | Set `POLY_PRIVATE_KEY` and `POLY_PROXY_WALLET` environment variables |
| **Invalid private key** | Ensure key is 64 hex characters (0x prefix optional) |
| **Order failed** | Check sufficient USDC balance in your wallet |
| **WebSocket errors** | Verify network/firewall settings allow WebSocket connections |
| **Builder API errors** | Verify Builder Program credentials are correct |
| **Import errors** | Ensure package is installed: `pip install -e .` |
